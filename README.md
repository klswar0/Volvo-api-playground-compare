# Volvo API Playground Companion App

A simple utility web application designed to demonstrate, test, and compare the official **Volvo Connected Vehicle API Sandbox** with the unofficial, dynamic **Volvo API Playground**.

This app was built primarily for developers, tech enthusiasts, and voters evaluating the project on **Stardance**.

##  Main Repository

The code for the simulated alternative environment, and full documentation can be found in the main repository:

 **[Volvo API Playground - Main Repository](https://github.com/klswar0/Volvo-API-playground)** 

---

### Tech used

*   **Backend:** Python / FastAPI
*   **Frontend:** HTMX, Tailwind CSS v4 (Dynamic Mode)
*   **Templates:** Jinja2
