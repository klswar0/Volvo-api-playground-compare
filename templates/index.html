Welcome to Volvo API Playground companion app

This site is made to compare the official Volvo API sandbox with the playground version

Feature / Aspect Official Volvo Sandbox Volvo API Playground
Test Vehicles Two, statically defined demo vehicles. Multiple predefined vehicles with diverse datasets and possibilities of creating custom vehicles.
Interactivity Static, read-only API response examples. Interactive environment the virtual car reacts dynamically to your API calls.
API Keys Single example shared with 2 predefined vehicles. Default API keys assigned to multiple varied sample vehicles.
Data Customization No modification allowed responses are always immutable. Highly customizable and flexible mock vehicle data.
Endpoint Scope Official production API fields and structures only. Additional utility endpoints for faster testing and development.
Scenario System No scenario system or driving simulation support. Built-in support for templates and scenarios Work in Progress.
Primary Purpose Simple API demonstration and structural overview. Complete development and testing environment for Volvo Connected Vehicle apps.
Security Manual access token generation needed (short-lived). Token validation, OAuth 2.0 flow or PKCE can be disabled for rapid testing.
